template<class Mixin, class RandomNumberGenerator = typename std::mt19937>
struct beluga::BaselineGeneration< Mixin, RandomNumberGenerator >

A particle generator.

An implementation of the ParticleBaselineGeneration named requirements.

Template Parameters
MixinThe mixed-in type. An instance m of Mixin must provide a protected method, m.self(). The return type of m.self() must satisfy the SensorModel named requirements.
RandomNumberGeneratorA random number generator, must satisfy the UniformRandomBitGenerator requirements.
