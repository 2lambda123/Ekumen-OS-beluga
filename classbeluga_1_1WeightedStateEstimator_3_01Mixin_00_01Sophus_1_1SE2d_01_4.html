Detailed Description

template<class Mixin>
class beluga::WeightedStateEstimator< Mixin, Sophus::SE2d >

Partial template specialization for weighted state estimator in 2D.

This class implements the EstimationInterface2d interface and satisfies Beluga named requirements: StateEstimator.

It's an estimator that calculates the pose mean and covariance using all the particles.
