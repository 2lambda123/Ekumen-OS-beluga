Detailed Description

template<class Mixin, class State>
class beluga::KldLimiter< Mixin, State >

Limiter policy that takes a number of particles defined by the KLD criteria.

An implementation of Beluga named requirements: Limiter.

See KLD-Sampling: Adaptive Particle Filters [1].

Template Parameters
MixinThe mixed-in type. Mixin::self_type::particle_type must exist and satisfy Beluga named requirements: Particle.
StateType that represents the state of a particle. It should match with particle_traits<Mixin::self_type::particle_type>::state_type.

Additionally, given:

  • P, the type Mixin::self_type::particle_type
  • p an instance of P
  • cp a possibly const instance of P
  • h an instance of std::size_t
