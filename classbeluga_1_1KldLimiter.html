Detailed Description

template<class Mixin>
class beluga::KldLimiter< Mixin >

Limiter policy that takes a number of particles defined by the KLD criteria.

An implementation of Beluga named requirements: Limiter.

See KLD-Sampling: Adaptive Particle Filters [1].

Template Parameters
MixinThe mixed-in type. Mixin::self_type::particle_type must exist and satisfy Beluga named requirements: Particle.

Additionally, given:

  • P, the type Mixin::self_type::particle_type
  • p an instance of P
  • cp a possibly const instance of P
  • h an instance of std::size_t

The following conditions must be satisfied:

  • The expression particle_traits<P>::cluster(cp) returns a std::size_t that represents the spatial hash of the particle cp.
  • The expression particle_traits<P>::cluster(p) = h is valid and assigns the cluster hash to the particle p.
    i.e. after the assignment h == particle_traits<P>::cluster(p) is true.
