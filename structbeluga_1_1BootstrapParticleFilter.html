Detailed Description

template<class Mixin, class Container>
struct beluga::BootstrapParticleFilter< Mixin, Container >

Base implementation of a particle filter.

BootstrapParticleFilter<Mixin, Container> is an implementation of the BaseParticleFilter named requirements.

Given T, the type named by Container::value_type,

  • particle_traits<T>::state_type must be the same as the state_type of the sensor and motion model.
  • particle_traits<T>::weight_type must be the same as the weight_type used in the sensor mode.
Template Parameters
MixinThe mixed-in type. An instance m of Mixin must provide a protected method, m.self(). The return type of m.self() must satisfy:
  • ParticleResampling
  • ParticleBaselineGeneration
  • ParticleSampledGeneration
  • MotionModel
  • SensorModel
ContainerThe particle container type. It must satisfy the ParticleContainer named requirements.
