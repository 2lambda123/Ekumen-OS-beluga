beluga named requirements: ParticleFilter

What an implementation of a particle filter in beluga should provide. This is satisfied for example by beluga::MCL<U, M, S, C> and beluga::AMCL<U, M, S, C>, for any valid U, M, S, C (see respective docs for detail).

Requirements

T is a ParticleFilter if:

  • T satisfies the BaseParticleFilter named requirements.
  • T satisfies the SensorModel named requirements.
  • T satisfies the MotionModel named requirements.
  • T satisfies the StateEstimation named requirements.
